# Advent of Code 2021

My solutions for [Advent of Code 2021](https://adventofcode.com/2021), written in Python.

## How to run

Install dependencies:

```
$ pip install -r requirements.txt
```

Run one:

```
$ python main.py -p 1
```

Run all:
```
$ for i in {1..25}; do python main.py -p $i; done
```
